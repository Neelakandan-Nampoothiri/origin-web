# Origin

A modern and responsive landing page built with React, Tailwind CSS, and Framer Motion, designed to showcase a premium digital brand experience with smooth animations, clean layouts, and modern UI principles.

## ✨ Features

- Modern and responsive design
- Smooth scroll-based animations
- Interactive user interface
- Mobile-first approach
- Reusable React components
- Optimized performance
- Clean and scalable project structure
- Tailwind CSS styling
- Framer Motion animations

## 🚀 Live Demo

https://originapp.onrender.com/

## 🛠️ Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Framer Motion
- JavaScript (ES6+)

### Deployment

- Render
